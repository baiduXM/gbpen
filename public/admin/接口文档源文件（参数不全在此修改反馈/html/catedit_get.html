获取栏目编辑数据


接口说明：

获取栏目编辑数据
地址 ../classify-info
参数
id(int)* 栏目ID  
返回值
err(int) 0 - 创建成功； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) id(int) 新增栏目id
name(string) 栏目名称
img(string) 首张预览图
type(int) 1-文字列表，2-图片列表，3-图文列表，
4-内容单页，5-功能，6-外链，7-微信功能，8-直达号功能，9-万用表单
module_key(string) 功能标识符，type==5,7,8有值（建议数据库无需实际存储，存储至url字段）
module_value(int) 功能模块的值，type==7,8有值（建议数据库无需实际存储，存储至url字段）
url(string) 外链地址
pc_show(int) 是否在PC下展示
mobile_show(int) 是否在手机浏览器下展示
wechat_show(int) 是否在微信下展示
p_id(int) 父级ID
keywords(string) 关键词，英文逗号分隔
description(string) 描述
page_content(string) 单页的HTML内容
