获取模板列表


模板管理页获取接口

接口说明：

获取模板列表
地址 ../template-list
模式 GET
参数
type(int)* PC模板or手机模板 1-PC模板；2-手机模板
current_page(int) 页码 默认1
per_page(int) 每页数量 默认8
search(string) 搜索模板 默认空 不搜索，对模板名称、编号进行搜索
classify_id(int) 行业分类筛选 默认空，不筛选
color(string,array) 颜色筛选 直接颜色标识符，或者标识符数组color[0]=red&color[1]=orange
返回值
err(int) 0 - 正常； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) total(int) 总数
per_page(int) 当前页数量
current_page(int) 当前页码
last_page(int) 最后一页页码
from(int) 起始索引
to(int) 结束索引
data(array) 0,1,2...(array) id(int) 模板id
serial(string) 模板编号
name(string) 模板名称（ini文件内定义）
img (string) 预览图
colors(array) 1,2..(array) id(str) 颜色风格，如red，blue...
value(str) 颜色十六进制代码
description(str) 颜色中文名称
is_selected(int) 是否选中
selected_style(int) 选中的颜色风格
created_at(int) 记录创建时间戳，不可修改
updated_at(int) 记录更新时间戳，不可修改
