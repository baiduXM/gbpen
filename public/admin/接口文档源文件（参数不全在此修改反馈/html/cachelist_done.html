已完成推送的缓存列表

此接口在开始刷新后频繁（约每隔1s）访问，直到刷新操作结束。

接口说明：

已完成推送的缓存列表
地址  
模式 GET
参数
     
返回值
err(int)

0 - 正常； 100-无缓存列表需要刷新；
其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) 0,1,2,3...(string) 待推送的静态文件名
如：article/1_2.html

参数示例数据：

{
	"err":0,
	"msg": "",
	"data": [
		"cat/1.html",
		"article/1_1.html"
	]
}
