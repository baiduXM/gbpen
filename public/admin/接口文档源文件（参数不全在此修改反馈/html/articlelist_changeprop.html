批量改变文章属性


接口说明：

批量改变文章属性
地址 ../article-batch-modify
模式 POST
参数
id(int,array)* 文章ID 直接id值，或者id数组id[0]=23&id[1]=25
set_star(int) 是否推荐 默认为空，不做修改。1-推荐，0-取消
set_top(int) 是否置顶 默认为空，不做修改。1-置顶，0-取消
set_pcshow(int) 是否PC显示 默认为空，不做修改。1-显示，0-取消
set_mobileshow(int) 是否手机显示 默认为空，不做修改。1-显示，0-取消
set_wechatshow(int) 是否微信显示 默认为空，不做修改。1-显示，0-取消
返回值
err(int) 0 - 全部操作成功； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) err==3001时有值
1,2...(array)		 部分成功的id数组

返回示例数据：

{
	id: 22
}
或
{
	id: [23,25]
}
