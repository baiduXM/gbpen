设置页面 保存

 

接口说明：

设置页面 保存
地址 customer-setting
模式 POST
参数
company_name(string)* 公司名称  
domain_pc(string)* PC域名 无需http://
domain_m(string)* 手机域名 无需http://
favicon(string) favicon图标地址  
logo_large(string)* LOGO大图地址  
logo_small(string) LOGO小图地址  
title(string) 网站标题  
keywords(string) 网站关键词 英文逗号隔开
description(string) 网站描述  
footer(string) 底部版权  
footer_script(string) 统计代码  
pc_num_per_page(int) pc每页个数 默认20
pc_num_pagenav(int) pc分页链接显示个数 默认0，不限
m_num_per_page(int) 手机每页个数 默认10
m_num_pagenav(int) 手机分页链接显示个数 默认0，不限
contactor(string) 联系人  
telephone(string) 固话  
mobile(int) 手机  
fax(string) 传真  
mail(string) 邮箱  
qq(int) QQ  
address(string) 公司地址  
返回值
err(int) 0 - 正常； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) null
