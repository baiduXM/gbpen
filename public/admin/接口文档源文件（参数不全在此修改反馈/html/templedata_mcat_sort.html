手机栏目修改排序


一次提交所有排序序列，此模式可兼容拖拽

接口说明：

手机栏目修改排序
地址 ../mhomepage-sortmodify
模式 POST
参数
indexlist(array)* 1,2,3...(array)
提交所有栏目的排序数据		 id(int) 栏目ID
index(int) 排序，越大越靠前
返回值
err(int)

0 - 全部操作成功； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array)

err==3001时有值
1,2...(array)

 部分成功的id数组

参数示例数据：

{
	indexlist: [{
		id: 23,
		index: 1
	}, {
		id: 24,
		index: 0
	}, {
		id: 25,
		index: 0
	}, {
		id: 26,
		index: 0
	}]
}
