接口文档说明


对接口文档的说明表格格式做说明。

接口说明：

image
(array)		 title(str) 图片标题
src(str) 图片地址
href(str) 图片链接

 

示例数据：

		image: {
			title: '测试图1',
			src: 'http://..../a.jpg',
			href: '{siteurl}/article/1.html',
		}

接口说明：

image
(array)		 0,1,2...(array) title(str) 图片标题
src(str) 图片地址
href(str) 图片链接

 

示例数据：

		images: [{
			title: '测试图1',
			src: 'http://..../a.jpg',
			href: '{siteurl}/article/1.html',
		}, {
			title: '测试图2',
			src: 'http://..../b.jpg',
			href: '{siteurl}/article/2.html',
		}]
