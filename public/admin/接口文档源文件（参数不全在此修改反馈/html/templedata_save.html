页面可编辑区域 保存


此页面内容的保存接口

接口说明：

保存页面可编辑区域列表内容
地址 ../homepage-modify
模式 POST
参数
type(int)* PC模板or手机模板 1-PC模板；2-手机模板
目前只支持PC

page(string)*

 页面标识符 index|list-text|content-news...（目前就只有index）

data(array)

title,slidepics...(?)

提交参数名与获取接口的变量名相同
可以仅提交有修改的变量

 type=="text"
(string)		 文本值，适用于标题文案等
type=="textarea"
(string)

文本值，适用于公告等

type=="image"
(array)		 title(str)* 图片标题
image(str)* 图片地址
link(str)* 图片链接
description(str) 图片描述，可传入\r\n
type=="images"
(array)		 0,1,2..(array)

格式同type=="image"
多张图片格式的数组

type=="list"
(array)

 id(int)* 选中的栏目ID
star_only(int)* 1-只读取推荐文章，0-不限
   
type=="navs"
(array)		 0,1,2..
(array)		 name(str) 导航名称
image(str) 导航图标地址
data(str) 导航数据
type==tel：值为电话号码
type==sms：值为手机号码
type==im：值例"000@53kf"，
@后为im类型标识符，@前为im的账号id
type==link：值为外链地址
type(str) 导航类型
tel|sms|im|share|link
enable(int) 导航是否启用
返回值
err(int) 0 - 正常； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) null
