添加/编辑文章


接口说明：

添加/编辑文章
地址 ../article-create
模式 POST
参数
id(int) 文章ID 如果提交此参数则是修改操作，否则新增
title(string)* 文章名称  
title_bold(int) 标题是否加粗 默认0
title_color(string) 标题颜色 默认空，十六进制颜色代码
c_id(int)* 栏目ID  
viewcount(int) 点击数 默认0
is_show(array) 站点展示位置
0,1,2...(array)		 值为 pc_show|mobile_show|wechat_show 分别对应PC、手机、微信下展示的栏目
如：is_show[0]=mobile_show&is_show[1]=wechat_show
content(string)* 文章内容HTML  
pubdate(int) 文章发布时间戳 默认当前时间，编辑下此参数无效，根据此来排序
返回值
err(int) 0 - 正常； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) id 新增/编辑文章ID值
