修改文章排序

此接口废弃，暂使用pubdate来识别文章排序

接口说明：

修改文章排序
地址  
模式  
参数
id(int,array)* 文章ID 直接id值，或者id数组id[0]=23&id[1]=25
action(string)* 如何移动 up|down|top|bottom|jumppage|jumpto
jump(int) 跳转值 action==jumppage：值为页码，必填
action==jumpto：值为强制设置的index排序索引值，必填
返回值
err(int) 0 - 正常； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) null

参数示例数据：

上移一条：
{
	id: 22,
	action: "up"
}
或，跳到第二页第一条
{
	id: [23,25],
	action: "jumppage",
	jump: 2
}
