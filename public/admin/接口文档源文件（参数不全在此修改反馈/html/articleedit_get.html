获取文章编辑数据


编辑文章界面，获取已有数据

接口说明：

获取文章编辑数据
地址 ../article-info
模式 GET
参数
id(int)* 文章ID  
返回值
err(int) 0 - 正常； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) id(int) 文章id
title(string) 文章名称
title_bold(int) 标题是否加粗
title_color(string) 标题十六进制颜色代码
img (array) 预览图（可能是多张）
viewcount(int) 点击数
pc_show(int) 是否在PC下展示
mobile_show(int) 是否在手机浏览器下展示
wechat_show(int) 是否在微信下展示
c_id(int) 栏目ID
content(string) 文章内容HTML
pubdate(int) 文章发布时间戳，可修改，根据此来排序
created_at(int) 记录创建时间戳，不可修改
updated_at(int) 记录更新时间戳，不可修改
