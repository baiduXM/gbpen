设置页面

 

接口说明：

设置页面
地址 customer-info
模式 GET
参数
     
返回值
err(int) 0 - 正常； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）
data(array) company_name(string)* 公司名称
domain_pc(string)* PC域名
domain_m(string)* 手机域名
favicon(string) favicon图标地址
logo_large(string)* LOGO大图地址
logo_small(string) LOGO小图地址
title(string) 网站标题
keywords(string) 网站关键词
description(string) 网站描述
footer(string) 底部版权
footer_script(string) 统计代码
pc_num_per_page(int) pc每页个数
pc_num_pagenav(int) pc分页链接显示个数
m_num_per_page(int) 手机每页个数
m_num_pagenav(int) 手机分页链接显示个数
contactor(string) 联系人
telephone(string) 固话
mobile(int) 手机
fax(string) 传真
mail(string) 邮箱
qq(int) QQ
address(string) 公司地址
