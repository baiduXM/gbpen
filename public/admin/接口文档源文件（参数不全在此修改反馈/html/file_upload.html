文件上传


接口说明：

文件上传
地址 ../file-upload
参数
file(file,array)* 上传目标文件 可能是单个文件，或者批量
target(string)* 上传目标位置

page_index|category|article|weixin|zhidahao
page_index：编辑首页内容下的图片（后期每个页面都有page_*）
category：栏目图片
article：文章图片
weixin：微信图片
zhidahao：直达号图片
返回值
err(int)

0 - 全部操作成功； 其他 - 错误（具体参见错误代码表）
msg(string) 错误提示信息（err为0时 值为空）

data(array)

 typeof(file)==file name 文件名
url 文件外链地址
typeof(file)==array

1,2,3...(array)
